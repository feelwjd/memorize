## CRM
- vocId : 12 자리, tmpId (임시vocId) : reqSysCd(sfa) 3자리 + 17 자리 = 20자리
- 한번 사용 된 tmpId 값은 재사용 불가.
- 사용할 API 
	- IF_VOC_LIST ; VOC 조회 - 목록
	- IF_VOC
